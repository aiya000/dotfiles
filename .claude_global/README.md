# .claude_global

Here is templates of Claude Code settings.
(NOTE: `.claude_global` is not general name.)

## Setup

```shell-session
$ [[ -d ~/.claude ]] || mkdir ~/.claude
$ ln -s ~/.dotfiles/.claude_global/CLAUDE.md ~/.claude/CLAUDE.md
$ ln -s ~/.dotfiles/.claude_global/settings.json ~/.claude/settings.json
$ ln -s ~/.dotfiles/.claude_global/commands ~/.claude/commands
```
